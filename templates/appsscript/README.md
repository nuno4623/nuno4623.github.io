# 앱스스크립트 제출형 — 설치 가이드 (한 번만)

학생 제출을 Google Sheets 로 받는 방식입니다. **한 번 세팅해두면 모든 제출형 워크시트가 공용으로 사용**할 수 있어요(워크시트별로 시트 탭이 자동 분리됨).

## 1. 스프레드시트 만들기
1. [sheets.new](https://sheets.new) 로 새 스프레드시트 생성
2. 주소창의 `https://docs.google.com/spreadsheets/d/`**`이부분`**`/edit` 에서 가운데 긴 문자열이 **스프레드시트 ID** 입니다. 복사해 두세요.

## 2. Apps Script 붙이기
1. 스프레드시트 상단 메뉴 **확장 프로그램 ▸ Apps Script**
2. 기본 `Code.gs` 내용을 지우고, 이 폴더의 **`Code.gs`** 내용을 붙여넣기
3. 맨 위 `SHEET_ID` 값을 위에서 복사한 ID 로 교체 → 저장(💾)

## 3. 웹앱으로 배포
1. 오른쪽 위 **배포 ▸ 새 배포**
2. 유형 선택(⚙️) ▸ **웹 앱**
3. 설정:
   - 실행 계정: **나(본인)**
   - 액세스 권한: **모든 사용자**  ← 학생이 로그인 없이 제출하려면 필수
4. **배포** ▸ 권한 승인(본인 계정) ▸ 나오는 **웹 앱 URL** 복사
   - URL 형태: `https://script.google.com/macros/s/…/exec`

## 4. 워크시트에 연결
- 제출형 워크시트(`worksheet.html`) 안의 `const ENDPOINT = '...'` 에 위 URL 붙여넣기
- `WS_ID` 를 워크시트마다 고유하게(예: `music5-samguk`) → 시트 탭 이름이 됩니다
- `N` 은 문항(textarea) 개수

## 확인
- 웹앱 URL 을 브라우저로 열면 `OK · 제출 수신 준비됨` 이 보이면 정상
- 워크시트에서 제출 → 스프레드시트에 새 행이 쌓이는지 확인

## 코드 수정 후 재배포
`Code.gs` 를 고쳤다면 **배포 ▸ 배포 관리 ▸ (연필) ▸ 버전: 새 버전 ▸ 배포** 를 해야 반영됩니다. (URL 은 그대로 유지됨)

---
💡 제출 확인 방식은 "보내고 완료 표시"(fire-and-forget)입니다. 크로스도메인 제약 없이 어떤 브라우저에서도 동작하도록 숨은 폼 전송을 사용합니다.
